# Wikipedia Table Scraper (Python)

This mini project demonstrates a basic web scraping workflow using Python.

## What it does
- Sends an HTTP request to a Wikipedia page (with a User-Agent header)
- Parses HTML using BeautifulSoup
- Extracts a comparison table into a pandas DataFrame
- Saves the result as a CSV file

## Tech stack
- requests
- beautifulsoup4
- pandas

## Output
- cloud_providers_comparison.csv
